# Easy Nonogram

内核版本：Nonogram 0.8

GUI版本：Easy Nonogram 0.7

数织，是通过线的提示数字编织符合数字的标记连块的小游戏，标记的对应元素具有地雷的性质，翻开即失败。

# 操作说明

1.开局通过鼠标或按数字键(1-5)或BIEGC选择难度

2.选择自定义难度时，按照`[行数] [列数] [雷数]`的形式输入，数字中间为空格，例如`24 24 288`，输入后回车

3.左键翻开，右键标记

4.鼠标左键或右键点击数字时切换数字暗淡/明亮

5.鼠标可拖动操作同线方块

6.Ctrl+滚轮调整大小

7.翻开所有非雷方块胜利，翻开雷失败

8.按空格执行标记校验，校验成功可胜利

9.按R重置地图

10.按Tab自动推进数织进度

11.终局时左键新游戏，右键关闭窗口
